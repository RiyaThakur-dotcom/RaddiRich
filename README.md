# RaddiRich ♻️💰

**Dustbin Nahi, Ye Tera ATM Hai! 🏧**

RaddiRich is a modern, mobile-first Web3-inspired clean tech platform built to revolutionize waste management in India. It empowers households to scan their everyday scrap, schedule pickups, and earn instant cash. This project demonstrates a production-ready, beautiful UI with PWA capabilities.

## 🚀 Features

- **PWA Ready:** Install the website directly onto a smartphone like a native mobile app.
- **AI Scanning Mockup:** Emulates taking a photo of scrap with predictive AI valuation (Google Gemini API mock).
- **Earnings Dashboard (Kamai):** Track your weekly recycling stats and money earned.
- **Nearby Recyclers Map:** Find the closest vendors with location tracking UI.
- **Full Responsive Design:** Beautiful interface built with Tailwind CSS v4, Framer Motion, and Lucide Icons.
- **Hinglish Toggle Concept:** Built for the native Indian audience with familiar, catchy taglines.

## 🛠️ Tech Stack

- **Frontend:** React 19 + Vite
- **Styling:** Tailwind CSS v4 + PostCSS
- **Routing:** React Router v7
- **Icons & Animations:** Lucide React + Framer Motion
- **Deployment:** Vercel (Configured)

## 🏃 Getting Started

1. Clone the repository
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the development server:
   ```bash
   npm run dev
   ```
4. Build for production:
   ```bash
   npm run build
   ```
